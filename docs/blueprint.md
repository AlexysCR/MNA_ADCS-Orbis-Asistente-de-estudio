# **App Name**: VoiceNoteAI

## Core Features:

- Voice Recording: Record audio notes directly within the app.
- AI-Powered Summarization: Automatically summarize voice notes using AI to extract key information.
- Topic Classification: Classify notes into predefined categories such as work, personal, study, etc using AI.
- Smart Task Tool: If the voice note is about a study topic or a task, generate a list of next steps and relevant references using a tool powered by generative AI.
- Note Storage: Securely store all voice notes and summaries.
- Search and Filter: Quickly find notes using search and filter by topic or date.
- Text Display: Displays a transcription of the voice note with formatting.
- Topic Clustering: If the same study topic coincides, then the voice note should be grouped to the already existing topic, keeping a record of the date and modification.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and focused feel.
- Background color: Light gray (#F5F5F5) for a clean and unobtrusive workspace.
- Accent color: Teal (#009688) to highlight important actions and elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look in both headlines and body.
- Use simple, clear icons for navigation and note categories.
- Prioritize a clean, card-based layout to organize notes and summaries efficiently.
- Subtle transitions when opening and closing notes, or when AI analysis is in progress.
